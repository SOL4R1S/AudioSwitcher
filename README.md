## AudioSwitcher (v1.0.0)

Nircmd를 활용하여 윈도우의 오디오 출력 장치를 빠르고 간편하게 변경해주는 프로그램입니다.
인터넷에 잠깐 보니 둘은 변환하기 쉬운데 셋부터는 그냥 코드짜는게 낫겠다 싶어서 ai 써서 만들었습니다.

## 주요 기능
- **자동 장치 감지:** 현재 연결된 스피커/헤드셋 목록을 자동으로 불러옵니다.
- **한/영어 지원:** 한국어(Korean) 및 영어(English) 인터페이스를 지원합니다.
- **설정 자동 저장:** Nircmd 경로와 언어 설정이 `json` 파일로 저장되어 다음 실행 시 편리합니다.

## 다운로드 및 실행 (For Users)
[Download Page](https://github.com/SOL4R1S/AudioSwitcher/releases/tag/v1.0.0) 에서 AudioSwitcher.exe 다운받으시면됩니다.

### 필수 준비물
이 프로그램은 소리 설정을 변경하기 위해 **Nircmd**가 필요합니다.
1. [NirSoft 공식 홈페이지](https://www.nirsoft.net/utils/nircmd.html)에서 `nircmd.exe`를 다운로드하세요. (맨 아래쪽 `Download NirCmd 64-bit` 추천)
2. 다운받은 `nircmd.exe`의 경로를 기억해두세요.

### 사용법
1. `AudioSwitcher.exe`를 실행합니다.
2. 처음 실행 시 **언어** (영어, 한국어)번호 입력하시고, **Nircmd 파일 경로**를 입력합니다 
    * (예: `C:\Users\Name\Downloads\nircmd.exe`).
3. 나타나는 목록에서 원하는 장치 번호를 누르면 즉시 변경됩니다.

### 이후 변경 예정
* 활성화되어있는 장치 중 입출력 동시 지원하는경우(헤드셋이나 뭐 입출력 동시 지원하는 사운드카드들)의 경우 출력장치 뿐 아니라 입력장치가 같이 튀어나오는 문제 변경(시기 미정)
* 출력장치만 만졌으니 입력장치 변경도..? (사실 저한테 수요는 없어서 안했습니다만 하게되면 변경할 장치고를 때 입력장치 변경창으로 가게끔 하나 만드는게 낫지 않나 생각중입니다)
